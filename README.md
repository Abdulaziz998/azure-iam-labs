# Azure IAM Labs

Hands-on Azure IAM labs demonstrating enterprise identity and access management using Azure RBAC, Managed Identity, Key Vault, Service Principals, and access control.

These labs simulate real-world Azure IAM engineering scenarios.

---

## Labs Covered

- Resource Group IAM (RBAC)
- Virtual Machine Access Control
- Managed Identity Authentication
- Azure Key Vault Access Control
- Custom RBAC Roles
- Storage Account IAM
- Service Principal Authentication

---

## Skills Demonstrated

- Azure RBAC configuration
- Identity-based access control
- Managed Identity configuration
- Service Principal authentication
- Azure resource access control
- Enterprise IAM architecture
